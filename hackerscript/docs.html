HackerScript

A fast, C-targeting systems language with modern syntax. Write clean, expressive code that compiles to portable C99 — no runtime, no GC, full control.

Compiles to C99 No Runtime Self-Hosted Arena Memory

Introduction

HackerScript (HCS) is a statically typed, compiled language that transpiles to C99. It gives you modern language ergonomics — objects, pattern matching, closures, a package manager — while staying close to the metal. Every HCS program becomes readable, portable C that you can inspect, optimize, and link against anything.

Design Goals

GoalApproach
Zero hidden costsNo GC, no reference counting — arena-based memory with explicit lifetimes
C interoperabilityNative FFI via extern, direct {} blocks for raw C injection
Readable outputGenerated C is clean, named, and debuggable with standard tools
Self-hostingThe HCS compiler is itself written in HCS

String Literals

String literals use square brackets: [hello world]. They produce a raw char* C string (null-terminated) usable with C functions. The Str type (from std/core) wraps them for dynamic use.

let greeting : ptr(u8) = [Hello, world!]

// Str (dynamic, heap-managed)
let mut s : Str = Str.from_cstr([Hello])
s.push(33)   // append '!'
s.println()
s.free()

Functions

Basic Function

fn add(a: i32, b: i32) : i32
  return a + b
end

// Call:
let sum : i32 = add(3, 7)

Void Function

fn print_line(s: ref(Str)) : void
  s.println()
end

Default Parameters & Multiple Returns

// Simulate multiple returns via out-params
fn divide(a: i64, b: i64, out_rem: ptr(i64)) : i64
  let _ptr : ptr(i64) = out_rem
  direct { *_ptr = a % b; }
  return a / b
end

let mut rem : i64 = 0
let q : i64 = divide(17, 5, &rem)
// q = 3, rem = 2

Passing Strings

// By reference (read-only): use ref(Str)
fn print_name(name: ref(Str)) : void
  name.println()
end

// By mutable reference: use ptr(Str)
fn uppercase(s: ptr(Str)) : void
  let up : Str = s.to_upper()
  s.free()
  // *s = up would require direct block
end

Recursive Functions

fn factorial(n: u64) : u64
  if n == 0
    return 1
  end
  return n * factorial(n - 1)
end

fn fib(n: u64) : u64
  if n <= 1
    return n
  end
  return fib(n - 1) + fib(n - 2)
end

  end

Objects & Methods

Defining a Struct

obj Point
  pub x : f64
  pub y : f64
end

  pub origin : Point
  pub width  : f64
  pub height : f64
end

Instantiation

let p : Point = Point { x: 1.0, y: 2.5 }

let rect : Rectangle = Rectangle {
  origin: Point { x: 0.0, y: 0.0 },
  width: 100.0,
  height: 50.0
}

Methods (impl block)

impl Point
  fn new(self: ref(Point), x: f64, y: f64) : Point
    return Point { x: x, y: y }
  end

  fn distance(self: ref(Point), other: ref(Point)) : f64
    let dx : f64 = self.x - other.x
    let dy : f64 = self.y - other.y
    return sqrt(dx*dx + dy*dy)
  end

  fn scale(self: ref(Point), factor: f64) : Point
    return Point { x: self.x * factor, y: self.y * factor }
  end
end

// Usage:
let a : Point = Point.new(0.0, 0.0)
let b : Point = Point.new(3.0, 4.0)
let d : f64 = a.distance(b)  // 5.0

Inline Methods in obj

obj Stack
  pub data : Vec
  pub size : u64

  fn new(self: ref(Stack)) : Stack
    return Stack { data: Vec.new(), size: 0 }
  end

  fn push(self: ref(Stack), val: u64) : void
    self.data.push(val)
    self.size = self.size + 1
  end

  fn pop(self: ref(Stack)) : u64
    self.size = self.size - 1
    return self.data.pop()
  end

  fn is_empty(self: ref(Stack)) : bool
    return self.size == 0
  end
end

let mut s : Stack = Stack.new()
s.push(42)
s.push(100)
write [popped value]    // pops 100

end

Enum with Data (Tagged Union)

enum Shape
  Circle(radius: f64)
  Rectangle(w: f64, h: f64)
  Triangle(base: f64, height: f64)
end

fn area(s: ref(Shape)) : f64
  match s
  when Shape.Circle(r) then
    return 3.14159 * r * r
  when Shape.Rectangle(w, h) then
    return w * h
  when Shape.Triangle(b, h) then
    return 0.5 * b * h
  end
end

Pattern Matching

Match on Integer

match status_code
when 200 then
  write [OK]
when 404 then
  write [Not Found]
when 500 then
  write [Server Error]
when _ then
  write [Unknown]
end

Match on Enum

enum Token
  Ident(name: Str)
  Number(val: i64)
  Eof
end

fn describe(tok: ref(Token)) : void
  match tok
  when Token.Ident(n) then
    write [identifier]
  when Token.Number(v) then
    write [number]
  when Token.Eof then
    write [end of file]
  end
end

Guards

match value
when n if n < 0 then
  write [negative]
when 0 then
  write [zero]
when n if n > 0 then
  write [positive]
end

OR patterns

match ch
when 65 | 69 | 73 | 79 | 85 then
  write [uppercase vowel]
when _ then
  write [other]
end

Pointers

Raw Pointers

let mut val : i32 = 42
let p : ptr(i32) = &val      // take address
let deref : i32 = *p         // dereference

// Pointer arithmetic via ptr_at()
let buf : ptr(u8) = some_buffer()
let byte_3 : u8 = ptr_at(buf, 3)
let byte_4 : u8 = *(buf + 4)

Pointer to Struct

let node_ptr : ptr(Node) = make_node()

// Access fields through pointer:
let val : i32 = node_ptr.value   // auto-deref
let next : ptr(Node) = node_ptr.next

ref vs ptr

TypeMeaningUse when
ref(T)Borrowed reference to T (read-only pointer)Function parameters that won't be stored
ptr(T)Owning or mutable raw pointer to THeap allocation, mutable out-params, linked structures

Extern & FFI

Calling C Functions

extern [c] header [stdio.h] {
  fn printf(fmt: ptr(u8), ...) : i32
  fn fflush(stream: ptr(u8)) : i32
}

extern [c] header [math.h] {
  fn sqrt(x: f64) : f64
  fn sin(x: f64) : f64
  fn pow(base: f64, exp: f64) : f64
}

// Usage:
let root : f64 = sqrt(144.0)   // 12.0

Linking a Library

extern [c] link [pthread] header [pthread.h] {
  fn pthread_create(
    thread: ptr(u64),
    attr: ptr(u8),
    start: fn(ptr(u8)) : ptr(u8),
    arg: ptr(u8)
  ) : i32
}

Extern Variables

extern [c] header [errno.h] {
  var errno : i32
}

Direct C Blocks

When you need to write raw C, use a direct {} block. The content is copied verbatim into the generated C. Use this for unsafe operations, assembly, or C-specific patterns.

// Write to a pointer without the HCS type system
let p : ptr(i32) = allocate_int()
direct { *p = 99; }

// Arena allocation
let mut node_ptr : ptr(Node) = nil
let sz : u64 = 32
direct { node_ptr = malloc(sz); }

// Memset
direct {
  alloc _buf : u8 * 1024
  memset _buf 0 1024
}
⚠ Warning Direct blocks bypass the type system. Use with care — wrong pointer arithmetic or missing null-checks will crash at runtime.

Str — Dynamic String

Str is the primary string type in HackerScript. It owns its buffer, supports efficient appending, and must be freed when no longer needed.

Creating Strings

let empty : Str = Str.new()
let from_literal : Str = Str.from_cstr([hello])
let with_capacity : Str = Str.with_cap(256)
let copy : Str = original.clone()

Building Strings

let mut s : Str = Str.new()

// Push a single byte
s.push(72)   // 'H'

// Push a string literal
s.push_str([ello, World!])

// Append another Str
s.append(other_str)

// Append a C string
s.append_cstr(some_cstr_ptr)

// Push formatted number
s.push_u64(42)
s.push_i64(-7)
s.push_f64(3.14)

Querying Strings

let len : u64  = s.len          // byte length
let b   : u8   = s.at(0)        // byte at index
let raw : ptr(u8) = s.buf       // raw pointer
let eq  : bool = s.eq(other)    // structural equality
let has : bool = s.contains_byte(47)  // has '/'
let idx : i64  = s.index_of_byte(46)  // find '.'

Transforming Strings

let trimmed : Str  = s.trim()
let lower   : Str  = s.to_lower()
let upper   : Str  = s.to_upper()
let sub     : Str  = s.slice(2, 8)    // bytes 2..8
let no_pfx  : Str  = s.strip_prefix([Hello])
let no_sfx  : Str  = s.strip_suffix([!])

Output

s.print()    // stdout, no newline
s.println()  // stdout + \n
s.eprint()   // stderr, no newline
s.eprintln() // stderr + \n

Memory

s.free()  // always free when done!
✓ Convention Always call s.free() before the owning scope exits. The HCS runtime uses arena allocation internally, but Str.buf is heap-allocated and must be freed manually.

Vec — Dynamic Array

Vec stores u64 values. Cast pointers to/from u64 to store any heap-allocated type.

let mut v : Vec = Vec.new()

// Push integers
v.push(10)
v.push(20)
v.push(30)

// Push a pointer (cast to u64)
let node : ptr(Node) = make_node()
v.push(node as u64)

// Access by index
let val : u64 = v.get(0)                   // raw u64
let ptr : ptr(Node) = v.get(1) as ptr(Node)  // cast back

// Mutate
v.set(0, 99)

// Iterate
let mut i : u64 = 0
while i < v.len
  let item : u64 = v.get(i)
  i = i + 1
end

// Pop last element
let last : u64 = v.pop()

v.free()
MethodSignatureDescription
Vec.new()→ VecCreate empty vector
v.push(val)(u64) → voidAppend element
v.get(i)(u64) → u64Element at index
v.set(i, val)(u64, u64) → voidSet element at index
v.pop()→ u64Remove and return last
v.lenfield u64Current length
v.is_empty()→ boolTrue if len == 0
v.free()→ voidFree backing storage

HashMap — Hash Table

HashMap maps Str keys to u64 values. Keys are stored by value (cloned on insert).

let mut map : HashMap = HashMap.new()

// Insert
map.set_cstr([name], some_str_ptr as u64)
map.set_cstr([count], 42)

// Check presence
if map.has_cstr([name])
  let v : u64 = map.get_cstr([name])
  let s : ptr(Str) = v as ptr(Str)
end

// Delete
map.del_cstr([name])

map.free()

I/O

Output

// Builtin write statement — prints a string literal
write [Hello, world!]
write []   // blank line

// Str methods for dynamic content
let mut msg : Str = Str.from_cstr([Result: ])
msg.push_u64(answer)
msg.println()
msg.free()

// Write to stderr
let err : Str = Str.from_cstr([error: something went wrong])

File I/O

// Read entire file into Str
let content : Str = read_file([data.txt])
write [file read]
content.free()

// Write Str to file
let mut out : Str = Str.from_cstr([Hello, file!])
write_file([output.txt], out)
out.free()

// Check file existence
if file_exists([config.toml])
  write [config found]
end

virus — Package Manager

virus is the official HackerScript package manager and project tool. It handles project creation, dependency management, and building.

Commands

virus init NAME

Create a new project with Project.toml and src/ structure

virus build

Build the current project. Compiles to .cache/, binary to build/

virus build --release

Optimized build with -O2

virus add NAME

Add a package from the registry to Project.toml

virus remove NAME

Remove a package from Project.toml

virus search QUERY

Search the registry for packages

virus info NAME

Show package details and available versions

virus fetch

Download all dependencies declared in Project.toml

virus emit-c

Transpile .hcs → C without compiling

virus check

Type-check without building

virus clean

Remove .cache/ and build/

virus version

Print virus version

Build Flow

# Default cmake flow:
virus build
#   1. hackerc cmake src/main.hcs --out-dir .cache/
#   2. cmake -S .cache -B build
#   3. cmake --build build
#   → binary in build/

# Standalone (no cmake):
virus build --standalone
#   1. hackerc build src/main.hcs -o build/myapp

Registry & Libraries

The official registry is at https://github.com/virus-repo/repository. Libraries are distributed as pre-built .a static archives.

Workspaces

Workspaces let you manage multiple related HCS projects as a single unit. A workspace root contains a workspace.toml and member projects in subdirectories.

Creating a Workspace

virus workspace init myworkspace
cd myworkspace

# Create member projects:
virus init server
virus init client
virus init shared

# Add them to workspace:
virus workspace add server
virus workspace add client
virus workspace add shared

workspace.toml

[workspace]
name = "myworkspace"
members = [
  "server",
  "client",
  "shared"
]

Building the Workspace

# Build ALL members in order:
virus workspace build

# Release build of all members:
virus workspace build --release

Workspace Commands

virus workspace init NAMECreate a new workspace directory with workspace.toml
